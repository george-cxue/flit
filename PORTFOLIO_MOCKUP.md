- **Card Background:** #F8FAFC (Light Gray)

### Dark Mode
- **Primary Color:** #6B9BFF (Light Blue)
- **Success (Gains):** #34D399 (Green)
- **Danger (Losses):** #F87171 (Red)
- **Card Background:** #1E293B (Dark Slate)
- **Border:** #334155 (Slate)
- **Text:** #ECEDEE (Light Gray)

## Interactive States

### League Tabs
- **Unselected:** White/Dark background, bordered
- **Selected:** Primary color background, white text
- **Hover:** Subtle opacity change

### Timeframe Buttons
- **Unselected:** Transparent with border
- **Selected:** Primary color fill, white text
- **Active:** Touch feedback

### Asset Cards
- **Default:** Card background with icon and metrics
- **Pressed:** Opens modal with educational content
- **Risk Badge:** Color-coded (Green/Yellow/Red)

### Stock Search Results
- **Default:** Listed with symbol, name, sector, price
- **Pressed:** Opens purchase modal
- **Price Change:** Color-coded positive/negative

### Chart
- **Portfolio Line:** Solid, primary color, 2.5px width
- **S&P 500 Line:** Dashed, gray, 1.5px width
- **Hover:** Tooltip with date and percentage values
- **Grid:** Subtle gray lines

## Typography

### Headings
- **Screen Title:** 32px, Bold
- **Card Title:** 18px, Semibold
- **Section Label:** 12px, Semibold, Uppercase, Letter-spacing

### Body Text
- **Large Values:** 36px, Bold (Portfolio value)
- **Medium Values:** 20px, Bold (Asset amounts)
- **Standard Text:** 14-16px, Regular
- **Small Text:** 12px, Regular (Labels, metadata)

### Financial Data
- **Positive Change:** Green, Bold
- **Negative Change:** Red, Bold
- **Neutral/Prices:** Theme text color, Semibold

## Spacing & Layout

- **Screen Padding:** 16px
- **Card Padding:** 16-20px
- **Card Border Radius:** 12px
